A plugin for Admins to execute Custom Rounds. Just type the command and choose round from the menu.
Following rounds are available in v1.3
- Knife Only Round
- Zeus + Nades Round
- All kind od weapons round (Pistol rounds are headshot only)
- WTF Round

# Commands
- !rounds
